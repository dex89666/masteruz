# MasterUZ 🔧

Professional Handyman Services Directory for Uzbekistan

## About

MasterUZ is a comprehensive directory of professional handyman services across Uzbekistan. The platform connects skilled professionals with customers needing various home repair and maintenance services.

## Features

- 🔍 **Search Functionality** - Easily search for services or specialists
- 📁 **Category Filtering** - Filter by service type (plumbing, electrical, carpentry, etc.)
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- ⭐ **Professional Listings** - Verified specialists with ratings and reviews
- 📍 **Location-Based** - Find services in your city across Uzbekistan

## Services Available

- **Plumbing** - Professional plumbing repairs and emergency services
- **Electrical** - Licensed electrical work and lighting installation
- **Carpentry** - Custom furniture, woodwork, and furniture assembly
- **Painting** - Interior and exterior painting services
- **Appliance Repair** - Home appliance diagnostics and repair
- **HVAC** - Air conditioning and heating system services
- **General Repairs** - All-around handyman and home maintenance

## Usage

Simply open `index.html` in your web browser to access the directory. Use the search bar to find specific services or the category dropdown to filter by service type.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Browse or search for handyman services

## Contributing

To add more service providers or categories, edit the `index.html` file and add new service cards following the existing structure.
